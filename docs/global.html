diff(a, b, sym) → {Object}

Generates a diff for the jsons a and b. Searches wikipedia with the query and if the page has an infobox parses inn, drugclass, cas, atc and the sum formula. If the first result doesn't have an infobox then the next pages in the wikipedia search will be analysed instead.
